# Project Bible

## Vision
Crystal Hunter RPG is a browser-based action RPG that blends simple arcade combat with light progression systems. The project is designed to be playable immediately in a browser while remaining easy to extend with more content later.

## Core Pillars
- Fast, readable gameplay loop
- Strong visual feedback and progression
- Lightweight implementation with no build step
- Local persistence for progression and high score

## Current Scope
- Single-page canvas game
- Player movement, combat, dash, skill shots, and combo attacks
- Enemy waves, crystal collection, gold drops, and shop upgrades
- Boss encounters, dungeon entry, quests, and autosave

## Target Audience
- Players who enjoy short browser games
- Developers exploring game systems in plain HTML/CSS/JavaScript
- Prototype creators who want a playable foundation quickly

## Success Criteria
- The game opens and runs in a browser without setup beyond loading the page
- Core loop is clear: move, fight, collect, upgrade, survive
- The experience feels cohesive and extensible
