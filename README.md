# Crystal Hunter RPG

A browser-based action RPG built in a single HTML file with canvas rendering, progression, combat, crystals, shop, inventory, boss encounters, quests, and autosave.

## Controls
- Move: WASD or Arrow Keys
- Attack: Left Click or Space
- Skill: Right Click or Q
- Dash: Shift
- Inventory: I
- Shop: B
- Dungeon: E
- Pause: Esc

## Features
- Mouse-based combat
- Mana, stamina, dash, combo attacks, invulnerability frames
- Weapon tiers from Wood to Dragon
- Crystal rarity rewards
- Inventory, shop, and quest log
- Boss encounters and dungeon entry
- Local save with autosave every 30 seconds
