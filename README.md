# Crystal Hunter - New R3F Project

This branch contains a fresh React + @react-three/fiber (R3F) scaffold implementing the requested 2D-facing 3D scene (MVP).

How to run locally:

1. npm install
2. npm run dev

Notes:
- Assets are loaded from provided remote URLs (ground, logo, player spritesheet).
- This is an initial MVP. I will continue implementing enemy spawn, attacks, boss patterns, item pickups, UI, and deploy config on request.
