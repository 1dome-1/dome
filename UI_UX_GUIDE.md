# UI/UX Guide

## Goals
- Keep the HUD readable at a glance
- Make the core loop obvious to first-time players
- Use clear color and contrast for health, mana, and progression

## Current Interface
- Top header with title and status pill
- Central canvas game area
- HUD overlay with HP, mana, stamina, level, and gold
- Inventory and shop panels

## UX Recommendations
- Maintain one primary action per screen
- Keep menus lightweight and dismissible
- Make feedback text visible and readable
