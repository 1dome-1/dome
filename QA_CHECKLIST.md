# QA Checklist

## Functional
- [ ] Game loads without console errors
- [ ] Player can move and attack
- [ ] Enemies spawn and respond to the player
- [ ] Crystals and gold rewards work
- [ ] Shop purchases and upgrades function
- [ ] Save/load works across reloads

## UX
- [ ] HUD is readable
- [ ] Menus are easy to dismiss
- [ ] Controls are discoverable
